# 🔨 画像移動しまし～ん v1.0.0

サムネ表示して移動したいやつぽちぽちして一括移動させるツール。
---

## ✨ 主な機能

- **シンプルな構造**: 移動元と移動先のフォルダを指定
                     画像の読み込み後、移動する画像を選択。
                     画像を移動ボタンで選択した画像が指定フォルダに移動するシンプル構造。

[詳しい使い方はこちら（説明書.pdf）](./画像付き説明書.pdf)

## 📦 動作環境・必要ライブラリ

- **OS**: Windows 10 / 11 (推奨)
- **Python**: 3.10 以上
- **依存ライブラリ**:PyQt6
```bash
pip install PyQt6
```
